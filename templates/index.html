About This Project

The Edge AI PPE Detection System is an intelligent worker safety monitoring platform designed for manufacturing industries and construction sites. Using Computer Vision and the YOLO object detection model, the system automatically identifies workers, safety helmets, safety vests, masks, and PPE violations in real time. The application provides live camera monitoring, image-based detection, an analytics dashboard, email alerts, PDF and Excel report generation, and a secure admin panel. It helps improve workplace safety, reduce accidents, and support compliance with industrial safety standards.
